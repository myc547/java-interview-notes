## 📚 分类说明

### 1. java基础（20+题）
- Java基本概念、数据类型
- 面向对象、接口抽象类
- 异常处理、泛型
- 反射、注解
- Java 8+新特性

### 2. JVM（20+题）
- 内存模型、类加载机制
- GC算法、垃圾回收器
- JVM调优、问题排查
- OOM问题分析
- 类加载器、双亲委派

### 3. MySQL（20+题）⭐ 包含高级
- 索引类型、存储引擎
- 事务隔离级别、MVCC
- 锁机制、慢查询优化
- 主从复制、分库分表
- Redo Log、Undo Log、WAL
- InnoDB 崩溃恢复

### 4. Redis（20+题）⭐ 包含高级
- 数据类型、持久化
- 过期策略、淘汰策略
- 主从复制、集群
- 分布式锁、缓存问题
- Redis 高级特性

### 5. 并发编程（10+题）
- 线程池、线程状态
- synchronized、Lock
- volatile、CAS
- AQS、并发工具类
- ThreadLocal、死锁

### 6. 分布式（10+题）
- CAP定理、BASE理论
- 分布式锁、幂等性
- 消息队列、限流
- 分库分表、一致性哈希

### 7. 设计模式（10+题）
- 创建型：单例、工厂、建造者
- 结构型：适配器、装饰器、代理
- 行为型：策略、观察者、模板方法

### 8. 系统设计（10+题）
- 秒杀系统、短链接
- 分布式ID、消息推送
- 评论系统、搜索建议
- 日志收集、高并发架构

### 9. 分布式事务（10+题）
- CAP、BASE理论
- 2PC、3PC、TCC
- Seata AT模式
- 消息事务、本地消息表

### 10. Spring Boot（14+题）
- 自动配置原理、@SpringBootApplication
- 启动流程、配置文件优先级
- 数据访问、JPA、多数据源
- 异常处理、拦截器、过滤器
- AOP、事务管理、定时任务、异步任务

### 11. Kafka（10+题）⭐ 包含高级
- 架构原理、分区策略
- 消息顺序性、消息不丢失
- 消费者组、offset管理
- Kafka 高级特性

### 12. RocketMQ（8+题）
- 消息类型、顺序消息
- 事务消息、消费模式
- 高可用架构

### 13. RocksDB（7+题）
- LSM树、写入流程
- Compaction、性能优化

### 14. Hazelcast（5+题）
- 分布式数据结构
- 分布式特性、使用场景

### 15. ElasticSearch（5+题）
- 倒排索引、分词
- 查询DSL、性能优化

### 16. Netty（5+题）
- NIO、BIO、AIO
- 零拷贝、Netty 架构

### 17. 操作系统（5+题）
- 进程与线程
- 内存管理、文件系统
- 中断与系统调用

### 18. 数字货币交易系统（5+题）
- 交易系统架构
- 订单管理、撮合引擎
